# GENDIFF
#### Compares two configuration files and shows a difference.

## Usage
To compare files:
* ```npx gendiff -h``` # to get help
* ```npx gendiff -V``` # to get version
* ```npx gendiff <first file> <second file>```  # to guess is even the number
